# 🔐 密碼產生器 Password Generator

一個用 Python 寫的安全密碼產生器，支援自訂長度、字元類型，並自動評估密碼強度。

## ✨ 功能特色

- 自訂密碼長度（建議 12~32 字元）
- 可選擇包含：大寫字母、小寫字母、數字、特殊符號
- 一次產生多組密碼
- 自動評估密碼強度（非常弱 ～ 非常強）
- 自動複製第一組密碼到剪貼簿

## 🚀 使用方式

```bash
python password_generator.py
```
